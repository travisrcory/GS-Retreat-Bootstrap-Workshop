Liferay Portal 6.2 contains a broad set of new features and capabilities. Liferay recommends that all users upgrade to this release so they can take full advantage of these important options.   2. License Keys

    An enterprise license key is required for Liferay Portal 6.2 Enterprise Edition. If you have not received or have a question about your license keys, please request your licenses through a LESA ticket or by emailing a request to: provisioning@liferay.com.

  3. Liferay Enterprise Support Application (LESA)

    LESA is your support lifecycle management console. LESA provides an exceptional user interface for simplifying the creation and management of support issues. Users can create, track and escalate issues through LESA's exceptional user interface. Its ability to issue multiple assignees also allows more people to work on a single ticket at the same time.

    1. LESA can be accessed from the "Access Tickets" icon on the left in the Customer Portal or directly from here. Further documentation is available in the LESA User Guide.
    2. In order to access LESA, you must have a valid Liferay.com account with access to the Liferay Enterprise Support Application and Customer Portal. If you do not have a Liferay.com account please create one from the Registration link on the top right menu at Liferay.com. If you do not have access to the Customer Portal, or if you have issues accessing LESA or issues creating tickets within LESA, please email provisioning@liferay.com
